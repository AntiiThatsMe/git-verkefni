1. Hvað gera eftirfarandi Linux skipanir?
    * cd - Fara í möppu
    * ls - Lista möppur og skrár
    * pwd - Núverandi vinnusvæði
    * mkdir - Búa til möppu

2. Hvað gera eftirfarandi Git skipanir?
    * git clone - Setur eintak af git geymslu á vélina þína
    * git status - Sýnir stöðu vinnumöppunar
    * git diff - Sýnir mun milli mismunandi stöðum í git safninu

3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?
    * git log - Birtir öll commits sem eru í repository 
    * git checkout - Vísir yfir á aðra grein
    * git branch - Sýnir lista yfir allar staðaltengdar greinar

4. Hvað er útgáfustýring (Version Control)?
    * Fylgist með *breyingum á skŕáum* eða *skráasafni* með tímanum.

5. Hverjir eru helstu kostir við að nota GIT?
    * Git fylgist með ***Öllum*** breytingum á skráum og repository með version control
    * Git auðveldar samstarf með því að gera mörgum forritum kleift að vinna að sama verkefni á sama tíma án þess að skrifa yfir vinnu hvers annars
